# Counter App With JavaScript

A simple, lightweight counter application built with HTML, JavaScript, and Tailwind CSS.

## Features

- **Increment**: Increase the counter value by clicking the `+` button.
- **Decrement**: Decrease the counter value by clicking the `-` button.
- **Responsive Design**: Styled with Tailwind CSS for a clean and modern look.

## Technologies Used

- **HTML5**: Structure of the application.
- **Tailwind CSS**: Utility-first CSS framework for styling (loaded via CDN).
- **JavaScript**: Logic for handling counter state and DOM updates.

## How to Run

1. Open `index.html` in any modern web browser.
2. Use the buttons to interact with the counter.
